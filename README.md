# Machine_Learning
This repository contains practical implementations of core Machine Learning concepts, including regression models, classification algorithms and optimization methods. The modules emphasize experimentation, comparison, and analysis of model performance.

## Repository Structure

``` text
Machine_Learning/
├── linear_regression_bgd/       # BGD implementations: single-variable and multivariate linear regression
├── regression_ols_vs_sgd/       # Regression models: OLS vs SGD
├── scaling_impact_regression/   # Feature scaling analysis: normalization and standardization in regression
├── simple_perceptron/           # Classification model: simple perceptron implementation
└── README.md                    # Main documentation and index
```
